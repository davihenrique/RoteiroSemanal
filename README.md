# Roteiro Semanal

Esta aplicação é um **roteiro semanal** desenvolvido para ajudar na organização e planejamento de atividades. Ela utiliza um arquivo `.json` para representar os dados, como horários, dias da semana e atividades, de forma estruturada e fácil de editar.

## Como funciona?

O roteiro é configurado a partir de um arquivo `info.json`, onde é possível definir os horários, dias e atividades. A aplicação lê este arquivo e apresenta as informações de forma organizada.
